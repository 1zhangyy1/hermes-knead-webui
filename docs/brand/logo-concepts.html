捏 / Knead — logo concepts

对齐 aura 品牌(墨 mark + 暖光),不再用旧陶土印章。四个方向,先看,后定。

A · 双指压点 (推荐)

墨色 squircle + 两个柔和压点 —— “捏”的两指动作;干净、现代、过目不忘,极小尺寸也认得。可叠一层极淡暖光呼应产品气场。

捏 Knead

A · 深色底

暗色下:墨块反白为暖白块 + 墨点,或保持墨块。暖光在深色上更像余烬,质感好。

捏 Knead

B · 压痕 (从初版精炼)

一团圆润墨块,一个拇指压痕(用暖光做凹痕高光)。保留“捏黏土”叙事,但去掉土气、改成墨+暖光。

捏 Knead

C · 捏字  ·  D · 纯字标

C:把「捏」字做成 mark,强调 扌(手)旁——文化感强,小尺寸较难。D:纯字标 “knead”,小写收紧 + 一个暖点收尾,Linear/Vercel 式,最简、最稳。
