# General AI

General AI is the default chat product for quick questions, writing, analysis, and files.

Its product UI is the base chat surface. It does not need a separate workspace unless the user explicitly shapes it into something more specialized.

## Product Shape

General AI demonstrates `chat_only`: the normal chat view is already a valid
product workspace. This is important because not every useful AI product needs
a custom interface. Some products are better as a configured identity, prompt,
skills, tools, suggestions, and files inside the default chat surface.

Keep this example small. If General AI starts needing a special panel, that is a
sign the user should shape a more specific product instead.
