# notion

SaaS dashboard aesthetic — clean, functional, productivity-tool styling.

## Design Aesthetic

Clean, functional SaaS interface aesthetic. Dashboard-inspired layouts with clear data hierarchy. Modern productivity tool look: structured, organized, trustworthy. Information-dense but never cluttered.

## Background

- Color: Light Gray (#F7F7F5) or Pure White (#FFFFFF)
- Texture: None — clean solid backgrounds only

## Typography

### Headlines
System-UI style sans-serif: "semi-bold, clean geometric sans-serif with slightly tighter letter-spacing — functional and legible, no personality distractions, like a well-made web app." Medium-to-semi-bold weight for emphasis.

### Body
Same family in regular weight: "regular-weight clean sans-serif optimized for screen reading — comfortable 1.5-1.6 line height, neutral, information-first."

## Color Palette

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| Background | Light Gray | #F7F7F5 | Primary background |

## Visual Elements

- Card-based layouts with subtle borders or very light shadows
- Clean data tables and metrics displays
- Progress bars and percentage indicators
- Tag and label chips (rounded rectangles)
- Icon-based navigation (outlined, 16–20px)
- Subtle dividers between sections
- Breadcrumb and hierarchy indicators

## Do

- Use card-based content organization
- Create clear data hierarchy with size and weight
- Apply subtle shadows and borders for card depth
- Keep layouts grid-aligned with consistent gutters
- Present metrics prominently with large numbers

## Don't

- Use decorative illustrations or photos
- Add gradients or complex backgrounds
- Create artistic or asymmetric layouts
- Use blob shapes or organic forms
- Mix many different accent colors

## Best For

Product demos, SaaS pitch decks, productivity tool showcases, metrics dashboards, feature walkthroughs, B2B presentations
