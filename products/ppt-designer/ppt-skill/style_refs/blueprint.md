# blueprint

Precise technical blueprint style — engineering diagram aesthetics for analytical presentations.

## Design Aesthetic

Clean, structured visual language borrowed from architectural blueprints and engineering schematics. Precise, analytical, and aesthetically refined. Information presented in grid-based layouts with engineering precision and technical credibility.

## Background

- Color: Blueprint Paper (#FAF8F5) — warm off-white that mimics engineering paper
- Texture: Subtle light grid overlay (#E5E5E5 lines), very faint engineering paper feel

## Typography

### Headlines
Bold weight clean sans-serif: "bold, wide-set geometric sans-serif with uniform stroke widths and tight letter-spacing — authoritative, no-nonsense." Think structured, technical, like dimension labels on a CAD drawing but large.

### Body
Elegant serif for body explanations: "a refined text serif with slightly bracketed serifs and comfortable x-height — academic and readable at small sizes." Clean, editorial quality.

## Visual Elements

- Precise lines with consistent stroke weights (1px, 2px, 3px system)
- Technical schematics and clean vector diagrams
- Connection lines use straight lines or 90-degree angles only
- Data visualizations: clean, minimal charts with axis labels
- Dimension lines and measurement indicators
- Cross-section style diagrams
- Numbered callout labels in circles
- Isometric-style box diagrams for architecture

## Do

- Maintain consistent line weights throughout
- Use grid alignment for all elements
- Keep color palette restrained: blue, navy, amber accents only
- Create clear visual hierarchy through scale differences
- Use geometric precision for all shapes

## Don't

- Use hand-drawn or organic shapes
- Add decorative flourishes or illustrations
- Use curved connection lines between nodes
- Include photographic elements
- Mix warm and cool colors outside the defined palette

## Best For

Technical architecture diagrams, system design, data analysis, process flows, engineering documentation, infrastructure presentations, API docs made visual
