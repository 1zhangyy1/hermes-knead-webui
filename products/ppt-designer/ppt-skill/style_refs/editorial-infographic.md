# editorial-infographic

Modern magazine-style editorial infographic — clear visual storytelling, complex ideas made digestible.

## Design Aesthetic

High-quality magazine explainer aesthetic. Clear visual storytelling that transforms complex information into digestible narratives. Clean illustrations, structured multi-section layouts, professional typography. Think Wired, The Verge, or high-end science publication spreads.

## Background

- Color: Pure White (#FFFFFF) or Light Gray (#F8F9FA)
- Texture: None or very subtle paper grain for print feel

## Typography

### Headlines
Bold display: "bold display sans-serif or display serif with strong visual presence — editorial sophistication at large scale. Think a Wired magazine section header: big, confident, clear."

### Section Headers
Semi-bold sans-serif: "semi-bold clean sans-serif for section headers — clear hierarchy distinction from body text, slightly smaller than headline."

### Body
Humanist sans-serif: "regular-weight humanist sans-serif optimized for reading — comfortable 1.6 line height, accessible, never competing with visuals."

## Visual Elements

- Clean flat illustrations (never photos)
- Structured multi-section layouts with clear zones
- Callout boxes for key insights
- Icon-based data visualization
- Visual metaphors for abstract concepts
- Flow diagrams with clear directional hierarchy
- Pull quotes and highlight boxes
- Section dividers and visual breaks

## Do

- Create clear visual narrative flow (reading path)
- Use structured multi-section layouts with explicit zones
- Include callout boxes for key insights
- Design visual metaphors for complex ideas
- Maintain magazine-quality polish throughout

## Don't

- Use photographic imagery
- Create cluttered dense layouts
- Mix too many visual styles
- Add decorative elements without purpose
- Use handwritten or organic type

## Best For

Technology explainers, research summaries, policy briefings, science communication, investigative content, educational deep-dives, thought leadership pieces, long-form data stories
