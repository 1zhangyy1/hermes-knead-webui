# bold-editorial

High-impact magazine editorial style — bold typography, dramatic contrast, commanding presence.

## Design Aesthetic

Strong visual impact at magazine cover level. Bold typography and dramatic contrast. Full-bleed color blocks and oversized text create commanding presence. Every slide feels like a premium publication cover.

## Background

- Color: Deep Black (#0A0A0A) primary, or Deep Blue (#0F172A) alternative, or Pure White (#FFFFFF) for light mode
- Texture: None — clean solid backgrounds

## Typography

### Headlines
Bold condensed display: "extra-bold, tightly-tracked condensed geometric sans-serif — oversized to dominate the slide, all-caps, maximum impact. Thick stroke, no hairlines. Like a bold news headline from a print broadsheet."

### Body
Clean neutral sans-serif: "medium-weight neutral sans-serif with high contrast against background — small, functional, recedes behind the headline. Never decorative."

## Visual Elements

- Strong typography as visual element itself — text IS the design
- Geometric shapes and bold color blocks as backgrounds
- Full-bleed solid color or pure black
- High contrast: almost nothing between black and white with ONE accent
- Minimal decoration — let content speak
- Dynamic diagonal lines or angles as dividers

## Do

- Use extreme scale contrast (huge headline, tiny body)
- Create bold color block compositions
- Let negative space create visual tension
- Make every slide feel like a magazine cover
- Use ONE accent color per slide maximum

## Don't

- Use soft or muted colors
- Add unnecessary decorative elements
- Create busy, cluttered layouts
- Use thin or delicate typography
- Mix multiple accent colors

## Best For

Product launches, keynote speeches, brand showcases, marketing presentations, investor pitches, high-stakes single-message slides
