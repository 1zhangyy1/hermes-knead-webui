# minimal

Ultra-clean keynote style with maximum whitespace and zen-like simplicity.

## Design Aesthetic

Maximum whitespace with minimal elements. Zen-like simplicity where every element earns its place. Premium, refined aesthetic suitable for executive audiences. Less is more — remove until nothing more can be taken away.

## Background

- Color: Pure White (#FFFFFF)
- Texture: None — absolute clean, no grain or patterns

## Typography

### Headlines
Clean geometric sans-serif: light to medium weight, generous letter-spacing, large scale. Describes visually as "a precise geometric sans-serif with perfect circular Os and uniform stroke widths, medium weight, widely tracked." Not bold, not condensed — restrained and elegant.

### Body
Same family as headlines in lighter weight. Minimal size contrast. Clean, airy feeling throughout. Small, unobtrusive, never competing with headline.

## Color Palette

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| Background | Pure White | #FFFFFF | Main slide background |
| Primary Text | Near Black | #1A1A1A | Headlines, body |
| Secondary Text | Medium Gray | #6B7280 | Captions, metadata |
| Accent | Single Brand Color | #2563EB | One accent only, used sparingly |
| Dividers | Light Gray | #E5E7EB | Subtle separators |

## Visual Elements

- Single accent color used for one element maximum per slide
- Thin hairline rules (1px) for separation
- Generous margins (minimum 15% on all sides)
- Center or left-aligned layouts
- Simple geometric shapes only when necessary
- No decorative elements
- Data visualizations in single color or grayscale

## Do

- Embrace empty space as a design element
- Use single accent color only, sparingly
- Keep text minimal (≤ 10 words per "beat")
- Create breathing room between elements
- Use scale to create hierarchy (title huge, body tiny)

## Don't

- Fill empty space with decoration
- Use multiple accent colors
- Add icons or illustrations unless essential
- Create dense information layouts
- Use heavy bold weights on everything

## Best For

Executive briefings, keynote presentations, premium brand communications, investor meetings, high-level strategy decks
