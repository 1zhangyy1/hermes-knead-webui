# scientific

Academic scientific illustration style — precise diagrams, proper labeling, textbook clarity.

## Design Aesthetic

Academic scientific illustration aesthetic for biological pathways, technical systems, and data diagrams. Clean, precise diagrams with proper labeling and clear visual flow. Educational clarity with professional polish. Think academic journal figures and textbook illustrations.

## Background

- Color: Off-White (#FAFAFA) or Light Blue-Gray (#F0F4F8)
- Texture: None or very subtle paper grain for print feel

## Typography

### Headlines
Formal serif: "bold-weight formal text serif with slightly bracketed serifs — authoritative, academic, like a journal article title. Conveys rigor and precision."

### Labels
Clean sans-serif: "regular-weight clean sans-serif for diagram labels and annotations — legible at small sizes, never decorative, consistent sizing for visual hierarchy."

### Body
Serif for paragraphs, sans-serif for bullets: "academic publication styling — serif body text for prose, clean sans-serif for structured lists."

## Visual Elements

- Horizontal structure bases (membrane, timeline, layer)
- Labeled modular components with distinct fill colors
- Flow arrows with directional clarity (no ambiguity)
- Numbered step sequences
- Key component callouts in labeled boxes
- Cross-section and pathway diagrams
- Data charts with fully labeled axes
- Process summary boxes

## Do

- Use precise, consistent line weights throughout
- Label ALL components clearly — no unlabeled elements
- Show directional flow with explicit arrows
- Create clear numbered sequences for processes
- Include units on all data values

## Don't

- Use decorative illustrations unrelated to content
- Create imprecise or artistic/organic diagrams
- Omit important labels to save space
- Use inconsistent visual language across diagram parts
- Mix illustration styles

## Best For

Biology lectures, chemistry presentations, medical education, research summaries, academic conferences, technical process documentation, data-heavy analytical slides
